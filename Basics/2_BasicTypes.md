# Haskellの基本的な型

## 基本型
ここではほかの言語にも存在するような、極めて基本的な型を紹介する。
| 型 | 内容 | 具体例 |
| ---- | ---- | ---- |
| Int | 整数 | `1` `2` `-3` |
| Double | 実数 | `1.0` `2.2` `-0.1` |
| Bool | 真偽 | `True` `False` |
| String | 文字列 | `"aaa"` `"bbb"` `"あああ"` |
| Char | 文字 | `'a'` `'b'` `'c'` |
| ( ) | 値なし | `()` |

Intに似たIntegerや、Doubleに似たFloatなどもあるが、ほとんどの場合において同じように使えるため、ここでは省略する。

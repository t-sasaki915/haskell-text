Haskell入門書テスト

1. はじめに

このテキストは、著者のHaskellの理解度を確かめることを主な目的として作成された。 また、著者は数学やコンピューターサイエンス、Haskellの専門家ではない。 このテキスト内に間違いを見つけた人や、より良い説明を思いついた人は、ぜひGitHubで提案してほしい。 このテキストが誰かのHaskell学習に役立つようなことがあれば、大変うれしく思う。

1.1 テキストを読むにあたって必要なもの

必須
  • Windowsパソコン (Windows 10以上、LinuxやmacOSは筆者がわからない)
  • PowerShellの使い方など中級レベルのパソコン知識
  • 高校レベルの数学知識 (決して微分積分などを行うわけではない)
あるとよい
  • C言語やPythonなどのプログラミング経験
  • LispやOCamlなどの関数型プログラミング経験
  • 圏論やラムダ計算の知識
C言語やPythonといったプログラミング言語の知識は、このテキストを読み進めるうえで役に立つことも多いが、Haskellはその前提知識を覆すような考え方が多い。
